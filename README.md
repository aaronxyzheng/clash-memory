# Clash Memory

A Clash Royale-themed memory card game built with React.

![Demo](demo-video.mov)

## How to Play

The idea is simple, just click a card you haven't seen before until you win or until you click on one you have clicked and you lose.

## Features

- 130+ Clash Royale card images to randomise from
- Card animations on each turn
- Tilt effect on hover
- Score and high score panel
- Win and lose screens with Clash Royale emotes
- Background music

```bash
git clone https://github.com/aaronxyzheng/clash-memory.git
cd clash-memory

npm run dev
```
